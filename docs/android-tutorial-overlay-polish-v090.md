# Android Tutorial Overlay Polish v0.9.0

## 목적

첫 전투의 7단계 튜토리얼을 Android 세로 화면에서 읽기 쉬운 게임형 안내 카드로 정리한다. 전투 요소를 단계별로 강조하면서도 안내 카드가 영웅 스킬 패널을 가리지 않도록 한다.

## 구현 내용

- 전체 화면을 어둡게 하고 현재 설명 대상만 금색 프레임으로 강조한다.
- `전투 안내 1/7` 형식의 진행 표시와 7칸 진행 바를 제공한다.
- 이전, 건너뛰기, 다음 버튼을 고정된 하단 영역에 배치한다.
- 7단계에서 `다음`은 `전투 시작`으로 바뀌며 완료 상태를 로컬 저장한다.
- 4단계는 영웅 스킬 패널을 강조하고 안내 카드를 전투 화면 상단에 배치한다.
- 7단계는 상단 골드/전투 상태 영역을 강조해 하단 안내 카드와 분리한다.
- `TutorialManager.Previous()`를 추가해 이전 단계로 돌아갈 수 있게 했다.

## 단계별 강조 대상

1. 크리스탈 방어 영역
2. 3개 전투 라인
3. 자동 전투 영웅
4. 영웅 스킬 패널
5. 웨이브 종료 후 룬 선택
6. 승리 보상과 다음 스테이지
7. 골드와 영구 업그레이드
