# RuneGate Defense Korean World Identity Guide

RuneGate Defense는 한국 출시를 우선하는 모바일 2D 픽셀 판타지 디펜스다. 고유 세계관의 중심 어휘는 `문지기`, `봉문`, `균열`, `재문`, `기록`이다.

## Core Terms

| 용어 | 의미 | UI 사용 예 |
| --- | --- | --- |
| 문지기 | 균열과 재문을 막는 영웅 집단 | 문지기 기록 |
| 봉문 | 열린 문을 닫고 유지하는 의식/결계 | 봉문 위험 기록 |
| 균열 | 적성이 흘러나오는 틈 | 균열 적성 기록 |
| 재문 | 닫힌 뒤에도 다시 열리려는 불안정한 문 | 재문 숲 |
| 기록 | 캐릭터, 적, 사건을 문서처럼 정리한 세계관 장치 | 전술 기록서 |

## Naming Rules

- `Hero Card`보다 `문지기 기록`을 우선한다.
- `Monster Card`보다 `균열 적성 기록`을 우선한다.
- `Boss Card`보다 `봉문 위험 기록`을 우선한다.
- `Rune Card`는 전투 UI에서는 유지할 수 있지만, 도감/설정에서는 `전술 기록서` 또는 `룬 기록`을 우선 검토한다.
- `Goblin Forest`는 파일명/호환명으로 남기되, 표시명은 `재문 숲` 계열을 사용한다.

## Tone

- 전쟁 영웅 서사보다 문서, 기록, 위험 보고서, 현장 대응 느낌을 우선한다.
- 워크래프트식 종족 전쟁이나 파이널판타지식 직업 판타지보다, 봉문 현장의 기록물 같은 질감을 우선한다.
- 한국어 UI는 짧고 단단한 명사구를 선호한다.

## Runtime Visual Split

- 전투 화면: 작은 픽셀 스프라이트, 높은 가독성, 제한된 색 수.
- 도감/기록 화면: 컨셉 카드 이미지, 세계관 문서 질감.
- 현재 컨셉 이미지는 `Assets/_Project/Art/ConceptSheets`에 남긴다.
- 향후 전투 스프라이트는 `Assets/_Project/Art/RuntimePixel` 아래에 별도 제작한다.
