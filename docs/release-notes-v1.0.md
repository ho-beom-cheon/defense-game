# RuneGate Defense v1.0 Release Notes

RuneGate Defense v1.0 is the first public-test Android candidate for the Goblin Forest prototype.

## Included

- 3-lane fantasy defense battle
- Stage 1-10 progression
- 6 MVP heroes
- 6 MVP monsters and Orc Warlord boss
- Three telegraphed phase-specific attack patterns for Grumbar
- 20 rune cards
- Gold rewards and permanent upgrades
- Local JSON save
- Tutorial overlay
- Scene-aware Menu/Battle BGM with independent BGM/SFX settings
- Knight and Goblin visual pipeline placeholders
- Android build menu and release checklist

## Not Included

- Server
- Login
- Cloud save
- Ranking
- Multiplayer
- Gacha
- Ad SDK
- Billing SDK
- Firebase
- Addressables
- Final art pass

## Test Focus

- Stage unlock progression
- Save/load stability
- Stage 10 boss wave
- Tutorial first-run flow
- Android APK build and device launch
