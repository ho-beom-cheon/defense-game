# Android Title Menu Polish v0.9

## 목적

Android 첫 화면을 단순 개발 메뉴가 아니라 실제 게임 진입 화면처럼 정리한다. 기존 스플래시 배경과 IMGUI 구조는 유지하면서 브랜드, 진행 요약, 주 행동, 설정, 파괴적 행동 확인 절차를 분리했다.

## 구현 내용

- 상단에 `룬게이트 디펜스`, `RuneGate Defense`, `봉문 전선을 지켜라` 브랜드 영역을 배치했다.
- 세로 화면에서는 명령 패널을 하단에 두고, 가로 화면에서는 브랜드와 명령 패널을 좌우로 분리한다.
- 새 저장에는 `새 전선 시작`, 실제 진행 데이터가 있는 저장에는 `전선으로 돌아가기`를 주 행동으로 표시한다.
- 진행 저장에는 보유 골드, Chapter 1 클리어 수, 현재 난이도를 요약한다.
- `새 전선`과 `저장 기록 초기화`는 화면 중앙 dim 모달에서 다시 확인한다.
- 저장 초기화는 기본 메뉴에서 제거하고 설정 화면 안으로 이동했다.
- BGM, SFX, 튜토리얼 재표시 설정은 기존 동작을 유지한다.

## 저장 판별 기준

`runegate_save.json` 파일은 첫 실행 초기화 과정에서도 생성될 수 있다. 따라서 파일 존재만으로 이어하기를 표시하지 않는다. 아래 항목 중 하나가 있으면 실제 진행으로 판단한다.

- 보유 골드
- 클리어 스테이지
- 마지막 선택 스테이지
- 인트로 또는 튜토리얼 완료

내부 stage id는 타이틀 화면에 노출하지 않는다.
